# Basic Command

## mysql 명령어

### mysql 실행

```
mysql -u root -p
```

### 데이터 베이스 리스트 보기

```
show databases;
```

### 데이터 베이스 선택

```
use database;
```

### 테이블 보기

```
show tables;
```

### 테이블 생성 명령어 보기

```
show create table table_name;
```

### 테이블 모든 컬럼 보기

```
show full columns from table_name;
```
